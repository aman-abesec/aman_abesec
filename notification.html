How to make Qrcode using python.

First of all install two package.
1.pip install pillow 2.pip install qrcode 

Method-1
import qrcode
qr=qrcode.QRCode(
version=5,
box_size=10,
border=5
)
data="""Write some thing"""
qr.add_data(data)
qr.make(fit=True)
img=qr.make_image(fill="black",back_color="white")
img.save("1.png")
